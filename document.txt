# Проект по онлайн голосованию

## Функциональные требования
- Поддрежка ролевой модели (Администратор, Пользователь)
- Авторизация пользователей
- Добавление задания на голосование Администратором
- Голосование пользователей (один пользователь может отдать только один свой голос на одно задание по голосованию
- Закрытие голосования и подсчет результатов

## Нефункциональные требования
- Язык программирования: JavaScript
- База данных: MySQL
